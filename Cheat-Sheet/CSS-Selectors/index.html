overview

CSS selectors are patterns used to select the element(s) you want to style.

There are many types of selectors, each with a specific purpose.

Here, we'll cover the most common selectors including:

  • basic selectors

Basic Selectors

Basic selectors are the most basic type of selectors.

They select elements based on their tag name.

For example, the following CSS rule selects all <p> elements:

p {
                  color: red;
                }

The basic selectors are as follows in order of specificity (starting with the least). NOTE that specificity controls which selector trumps others:

  1. universal
  2. type
  3. class
  4. id
selector description HTML code | CSS code
* (universal) selects all elements <*> | * {}
type selects an element by its tag name <p> | p {}
.class selects an assigned element class <p class="class"> | .class {}
#id selects an assigned element id <p id="id"> | #id {}

Pseudo-element

The ::before pseudo-element is used to insert content before the content of an element.

It is often used to add cosmetic content to an element, by using the content property.

For example, the following CSS rule adds a red exclamation point before the content of all <p> elements:

p::before {
                  content: "!";
                  color: red;
                }

The ::after pseudo-element is used to insert content after the content of an element.

It is often used to add cosmetic content to an element, by using the content property.

For example, the following CSS rule adds a red exclamation point after the content of all <p> elements:

p::after {
                  content: "!";
                  color: red;
                }
